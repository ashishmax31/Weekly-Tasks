What we learned?

Apart from learning the fact that our search for the real world "Elrich Bachman" has ended,We learned the following stuff:

  • ssh:Securely access any server in the planet.
  • scp:Copy files from one system to another.
  • grep:Command line text filter.
  • curl:Transfer data from or to a server.
  • wc:Word count.
  • youtube-dl:Download videos from the terminal.
  • pwd:Print the current working directory.
